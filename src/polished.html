Project Summary

In order to study the Aplha diversity of the Narrow River in Rhode Island, samples from the River and Basin were taken for two years at varying locations and depths for metagenomic sequencing. 16S rRNA gene copies, richness of taxonomic groups, and dissimilarity Matrices (NMDS) were studied post sequencing to determine the diversity drivers within the ecological community. This project was completed in RStudio with traceable steps and workflow from R project based navigation.

Creating metagenomic sequencing object using “phyloseq” package

make_metagenomeSeq = function(physeq) {
  require("metagenomeSeq")
  require("phyloseq")
  if (!taxa_are_rows(physeq)) {
    physeq <- t(physeq)}
  OTU = as(otu_table(physeq), "matrix")
  ADF = AnnotatedDataFrame(data.frame(sample_data(physeq)))
  TDF = AnnotatedDataFrame(data.frame(OTUname = taxa_names(physeq), row.names = taxa_names(physeq)))
  MGS = newMRexperiment(counts = OTU, phenoData = ADF, featureData = TDF)
  MGS = cumNorm(MGS)
  return(MGS)
}
